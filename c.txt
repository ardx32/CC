Implementation (HTML/JS):

Create an index.html file with the following script layer.
<!DOCTYPE html>
<html>
  <head>
    <title>Map Implementation</title>
    <script src="https://maps.googleapis.com/maps/api/js?key=YOUR_API_KEY&callback=initMap" async defer></script>
    <script>
      function initMap() {
        // System coordinates centering
        var location = {lat: 19.0330, lng: 73.0297}; 
        var map = new google.maps.Map(document.getElementById('map'), {
          zoom: 12,
          center: location
        });
        var marker = new google.maps.Marker({
          position: location,
          map: map
        });
      }
    </script>
  </head>
  <body>
    <h2>REST API Integrated Map</h2>
    <div id="map" style="height: 500px; width: 100%;"></div>
  </body>
</html>

AWS Simple Workflow Framework (SWF) - Hello World
AWS SWF orchestrates complex, distributed workloads. 
It splits logic into Activities (the tasks) and Workflows (the coordination).

Conceptual Code Structure (Requires AWS Java SDK):
Note: This utilizes the Flow Framework paradigm.

Activity Interface & Implementation:

Java
@Activities(version = "1.0")
@ActivityRegistrationOptions(defaultTaskScheduleToStartTimeoutSeconds = 300)
public interface GreeterActivities {
    void printHello(String name);
}

public class GreeterActivitiesImpl implements GreeterActivities {
    @Override
    public void printHello(String name) {
        System.out.println("Hello, " + name + "!");
    }
}
Workflow Interface & Implementation:

Java
@Workflow
@WorkflowRegistrationOptions(defaultExecutionStartToCloseTimeoutSeconds = 300)
public interface GreeterWorkflow {
    @Execute(version = "1.0")
    void greet(String name);
}

public class GreeterWorkflowImpl implements GreeterWorkflow {
    private final GreeterActivitiesClient client = new GreeterActivitiesClientImpl();

    @Override
    public void greet(String name) {
        client.printHello(name); // Workflow coordinates the execution of the Activity
    }
}
